Необходимо на странице "Управления паролями"
* Включить автоматическое создание паролей
* Включить reCAPTCHA, указать ключи
* Настроить Шаблоны E-mail (в пакете locale_ru настраивается автоматически)

На странице "Настройка E-mail"
* настроить почту
* установить галочку "Разрешить пользователям использовать данную учётную запись для отправки исходящей почты"

Пакетом устанавливается
* хук прикрепления роли по умолчанию
* настройки конфигурации
* тема для личного кабинета SuiteR-PC
* темы, которые не заканчиваются на '-PC', отключаются в конфиге

Переустанавливать только через spm.
